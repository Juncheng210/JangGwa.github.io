Page+Hexo免费搭建酷炫个人博客

发表于 2016-07-14   |     |   阅读次数

前言

在我的理解，个人博客是第二份简历，甚至比简历还重要；而且个人博客可以记录很多东西，它可以永久存留。

说实话，搭建这个博客还是挺艰辛的，经历了各种错误，各种找资料，踩过各种坑，终于打造好了我的博客。好吧，废话不多说了，开始吧。先放自己的blog看下效果

搭建

1.安装Node
到Node.js官网下载相应平台的最新版本，一路安装即可。
2.安装Git
安装Xcode自带有Git。
3.申请GitHub
申请一个GitHub账号，配置好SSH Keys。

4.安装hexo：

$ sudo npm install -g hexo

5.Node和Git都安装好后新建个文件夹blog，然后进入blog

$ cd blog

6.初始化
然后，执行init命令初始化hexo到blog目录

$ hexo init

7.生成静态页面
cd 到你的blog目录，执行如下命令，生成静态页面至hexo\public\目录。

$ hexo g
执行完执行

$ hexo d

8.本地启动
启动本地服务，执行完下面命令浏览器输入localhost:4000就可以看见你的个人博客了

$ hexo s

9.配置Github
新建Repository，与你用户名对应的仓库，仓库名必须为your_user_name.github.io。比如我的JangGwa.github.io

10.建立关联
现在我们需要打开blog文件夹里的_config.yml文件，来建立关联,翻到最下面，改成我这样子的。要注意的是，该文件格式要求极为严格，缺少一个空格都会导致运行错误。小提示：不要用Tab缩进，用两个空格符。

11.执行如下命令才能使用git部署

$ npm install hexo-deployer-git –save

12.完工

$ hexo g

$ hexo d
然后再浏览器中输入http://JangGwa.github.io/就行了，我的github的账户叫JangGwa,把这个改成你github的账户名就行了

最后

希望大家能打造一个满意的个人博客，做更好的自己~.~
有什么问题也可以问我-.-
